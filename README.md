### 切换节点方法 

账号端口、密码不变，切换节点（节点≈服务器≈IP≈x.x.x.x)，普通用户换普通节点、精品用户换精品。换后重开连接开关，过30秒钟，重启浏览器测试。

**一个节点约等于一条车道，账号约等于道上的小汽车。每个车道都有堵车、车祸的可能，才需要切换车道。**
